# Security Policy

## Supported Versions

Only the latest release is supported with security updates.

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |
| Older   | No        |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly by [opening a private security advisory](https://github.com/thijsvos/hnt/security/advisories/new) instead of a public issue.

You can expect an initial response within 48 hours.
